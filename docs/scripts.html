Scripts Overview

risk-python contains 5 Python scripts and 10 SQL scripts organized into morning and evening workflows. There is no API — scripts are run manually or via Windows batch (.bat) files on a schedule.

Morning Scripts

Run_morning_ThaiBMA.py — Part 1

Logs into ThaiBMA, downloads Morningstar Benchmark, then downloads multiple bond index files.

StepActionTarget URL / Path
1Copy Morningstar Benchmark from mapped drive\\172.16.21.100\Risk$\Morningstar Benchmark\
2Selenium: Open Chrome, navigate to ThaiBMA loginhttps://www.ibond.thaibma.or.th/login
3Login: fill username/password XPath fields, click submitXPath selectors (hardcoded)
4Navigate to ZRR index page, trigger download/zrr-index
5Navigate to Short-term index, trigger download/shortterm-index
6Navigate to Bond index, trigger download/bond-index
7Navigate to MTM Corp index, trigger download/mtm-corp-index
8Navigate to Composite index, trigger download/composite-index
9Navigate to Corp ZRR index, trigger download/corp-zrr-index
10Copy downloads to network share\\w2fsspho101.lhfund.net\FM-RI$\risk\98.Data File\

Retry Logic

Login retries up to 360 iterations (0.5s sleep) = up to 3 minutes. Download wait: polls for .crdownload files with 10-second timeout. Chrome download dir: logfile_formorning\From_load\ on network share.

Chrome Options

--start-maximized
--force-device-scale-factor=0.35  # zoom out for full-page visibility
--high-dpi-support=0.35
download.default_directory = network_share\From_load

Run_morning_ThaiBMA_part_2.py — Part 2

Separate script (same structure as Part 1) for additional ThaiBMA index downloads. Reuses same Chrome setup, credentials, and network share paths.

Runs two Excel VBA macros and sends the GPO performance email. This is the canonical production evening script.

1Open Excel via win32compower_automate_for_Afternoon.xlsm on network share
2Run VBA macro Create_AfternoonMacro processes afternoon data; workbook closed without save
3Open Benchmark.xlsmNetwork share path
4Run VBA macro Module1.eveningUpdates Benchmark figures
5Read GPO-FIXED ExcelP:\Bloomberg\Management Fee for PVD\..._GPO-FIXED..._REVISED_RATE.xls — sheet: Benchmark - PI, header row 4
6Read GPO-EQ ExcelP:\Bloomberg\Management Fee for PVD\..._GPO-EQ..._REVISED_RATE.xls — sheet: Benchmark - PI
7Send HTML email via OutlookTo: risk@lhfund.co.th; operation@lhfund.co.th | Subject: อัพเดท GPO

Email Content

Subject: อัพเดท GPO

Environment Setup

Prerequisites

  • Windows 10/11
  • Microsoft Office (Excel + Outlook) installed and licensed
  • Anaconda Python 3 installed at C:\ProgramData\anaconda3\
  • Google Chrome installed
  • Network drive access to \\w2fsspho101.lhfund.net\FM-RI$\ and \\172.16.21.100\Risk$\
  • SQL Server access (LHF_SYSTEM, LHF_PERFORMANCE, INV_LHF databases)

Credential Setup

# Create PP.env on Desktop (C:\Users\[username]\Desktop\PP.env)
user=<ThaiBMA_username>
pass=<ThaiBMA_password>
🔒 Security: Never commit .env files

scripts/.env contains plaintext credentials. Ensure .gitignore excludes all *.env files before the first git push.

Install Dependencies

cd C:\Users\[username]\AppData\Local\anaconda3\Scripts
pip install selenium webdriver-manager pandas openpyxl python-dotenv pywin32

Running the Scripts

# Morning (double-click or run in CMD):
python Run_morning_ThaiBMA.py
python Run_morning_ThaiBMA_part_2.py

# Evening:
python GPO.py

# Or use the provided .bat files in file dib/
